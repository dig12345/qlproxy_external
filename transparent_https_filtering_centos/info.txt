Diladele Web Safety for Squid Proxy Server
----------------------------------------------------------------------------------------------------

Diladele Web Safety for Squid Proxy Server is an ICAP server that integrates with existing 
Squid proxy server and provides rich content and web filtering functionality to sanitize 
Internet traffic passing into internal home/enterprise network. It may be used to block illegal 
or potentially malicious file downloads, remove annoying advertisements, prevent access to 
various categories of the web sites and block resources with explicit content.

The application is easily deployed and managed, requires minimal external dependencies, very 
robust and runs with excellent performance. Supports all major Linux distributions (Ubuntu, 
RedHat, Debian, CentOS), FreeBSD, Raspberry PI, Beagle Bone Black and (in future) Apple Mac OX.
